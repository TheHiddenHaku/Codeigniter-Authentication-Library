QuickAuth Authentication Library

QuickAuth is an extremely flexible and easy to use authentication library for Codeigniter

Features:

  • Login
  • Logout
  • Logged In
  • Register
  • Restrict
  • Retrieve Lost Password

Initializing The Library

Within a text string, highlight variables using <var></var> tags, and highlight code using the <dfn></dfn> tags.

Like most other classes in CodeIgniter, the FTP class is initialized in your controller using the $this->load->library('quickauth') function: $this->load->library('quickauth'); Once loaded Quickauth functions will be available using: $this->quickauth

Usage Examples

In this example the library checks if the user is logged in and returns a boolean.

		$this->load->library('quickauth');

		if ($this->quickauth->logged_in() === TRUE)
		{
			// Show the Username of the logged in user.
			echo $this->session->userdata('username');
		}

Login

$this->quickauth->login()

The Login function accepts 3 arguements which correspond to the username and password from a submitted form and a target location to redirect too.

$this->quickauth->login('$username, $password, $redirect = NULL')

Logout

$this->quickauth->logout()

Logs the user out by destroying all session variables.

$this->quickauth->logout()

Logged In

$this->quickauth->logged_in()

Checks if the user is logged in and then returns a Boolean result (TRUE/FALSE) to indicate whether the user is logged in or not.

$this->quickauth->logged_in()

Register

$this->quickauth->register()

The Register function accepts 3 variables for username, password and email from a form that has been submitted to the controller.

$this->quickauth->register('$username, $password, $email')

Restrict

$this->quickauth->restrict()

Used for restricting users access to certain controllers and functions. The variable passed should be the Group ID of the group you wish to have access to the page. That group and above will then be able to access the page. Other users will see You do not have sufficient privileges to access this page.

$this->quickauth->restrict('$group_id')

Retrieve Password

$this->quickauth->retrieve_password()

This function will email a user a newly generated nine character alpha numeric password. The submitted variable should be a username or email submitted from a form.

$this->quickauth->retrieve_password('$data')

Activate User

$this->quickauth->activate_user()

This function will activate the account tied to the Username that is passed to it. It will return true if successful or 0 if there was a problem.

$this->quickauth->activate_user('$data')
