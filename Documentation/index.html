QuickAuth Authentication Library V2.1

Installling The Library

The QuickAuth authentication library is extremely easy to install and use for Codeigniter users of varying different experience levels.

In order to install the library download the code from GitHub and then extract the library and helper folders to your application folder within your existing Codeigniter installation.

There are two methods of loading QuickAuth and Ui Helper, the first is to load them individually using $this->load->library('quickauth'); and $this->load->helper('ui'); in any controller that you wish to use them.

The second methid is to add them to the library and helper arrays located in ./application/config/autoload.php by adding the quickauth and ui handles repectively to each array. I recommend this approach as an authentication library if used often throughout most applications and loading it individually could become quite annoying on larger applications.

Encrypt

Encrypt a string (usually a password) ready for use within the library. Uses SHA1 encryption against the Codeigniter encryption key

$this->quickauth->encrypt($string);

Logged In

Check to see if a user is logged in. If not then don't return anything.

$this->quickauth->logged_in();

Login

Log a user in using the supplied username and password combination.

$this->quickauth->login($username/$email, $password);

Logout

Log a user out by destroying their session, then set a ui message and return.

$this->quickauth->logout();

Register

Register a user. Data should be supplied in an array format using the structure specified below.

		$userdata = array(
		    'username' => $username,
		    'password' => $password,
		    'firstname' => $firstname,
		    'lastname' => $lastname
		);

		$this->quickauth->register($userdata);

Restrict

Restrict a controller to a user group, logged in users, or exclude the function from an existing restriction.

$this->quickauth->restrict($group_name);

User

Get the data on a user from the user table. Also parse their full name into $data['name'] for convinience.

$this->quickauth->user(); $this->quickauth->user($user_id);
